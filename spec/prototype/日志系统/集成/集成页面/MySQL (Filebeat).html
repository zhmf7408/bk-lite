概述

MySQL 集成使用 Filebeat 的 MySQL 模块，自动采集和解析 MySQL 数据库的错误日志和慢查询日志。通过结构化的日志数据，帮助您快速定位数据库问题、优化查询性能。

功能特性

📊

自动解析

内置 MySQL 日志解析规则，自动提取结构化字段

⚡

实时采集

增量采集日志，低延迟传输，支持断点续传

🔍

慢查询分析

解析查询时间、锁等待时间、扫描行数等性能指标

🚨

错误追踪

快速识别数据库错误、警告，支持按错误码筛选

解析字段

错误日志和慢查询日志将被解析为以下结构化字段：

错误日志字段

字段名 类型 说明
@timestamp date 日志时间戳
mysql.error.level keyword 日志级别（Note/Warning/Error）
mysql.error.code long MySQL 错误码
mysql.error.message text 错误消息内容

慢查询日志字段

字段名 类型 说明
mysql.slowlog.query_time.sec float 查询执行时间（秒）
mysql.slowlog.lock_time.sec float 锁等待时间（秒）
mysql.slowlog.rows_examined long 扫描行数
mysql.slowlog.rows_sent long 返回行数
mysql.slowlog.query text SQL 语句
mysql.slowlog.user keyword 执行用户
mysql.slowlog.host keyword 客户端主机
