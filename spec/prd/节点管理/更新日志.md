# 节点管理 PRD 更新日志

## 2026-04-06（v1.4）

- 文档版本升级为 `v1.4`（日期 `2026-04-06`）。
- 校准安装包口径：明确重复上传限制生效于“操作系统 + 对象 + 版本”维度，避免将跨操作系统版本误写为冲突。
- 补充配置初始化规则：明确节点首次纳管时会为默认运行的采集器初始化预置主配置，已有自定义主配置时不覆盖。
- 细化令牌规则：明确安装脚本令牌为 30 分钟 / 5 次，安装包下载令牌为 10 分钟 / 3 次。

## 2026-03-30（v1.3）

- 文档版本升级为 `v1.3`（日期 `2026-03-30`）。
- 校准节点展示口径：移除“节点标签展示”表述，改为当前页面已落地的组织归属、活跃状态、安装方式、节点类型、控制器状态、采集器状态与版本信息展示。
- 补充变量安全规则：明确密文类型变量默认脱敏展示，编辑时未显式输入新值则保留原值。
- 收紧控制器批量操作口径：明确远程安装/卸载按同操作系统且安装方式一致的节点批次处理，Windows 节点走手动安装。

## 2026-03-23（v1.2）

- 文档版本升级为 `v1.2`（日期 `2026-03-23`）。
- 校准变量管理口径：移除“变量批量删除”的产品能力表述，保留当前页面已落地的分页、搜索、新增、编辑、删除能力。
- 校准节点删除口径：明确节点手动删除仅移除当前纳管记录；若同一节点后续继续由 Sidecar 正常上报，将重新纳入节点清单。
- 补充采集器运行规则：明确节点采集器执行启动或重启时，若缺少主配置且采集器提供默认模板，系统会自动补建默认主配置后执行操作。
- 收紧可扩展性表述：明确当前口径为采集器对象与安装包版本治理可扩展，控制器按内置对象与多操作系统维度维护。

## 2026-03-18（v1.1）

- 文档版本升级为 `v1.1`（日期 `2026-03-18`）。
- 校准信息架构口径：一级导航更新为“云区域 / 组件库 / 控制器”，补充节点采集器详情抽屉、组件包管理页与控制器包管理页入口。
- 校准环境管理口径：明确当前环境页提供容器部署脚本生成与复制，K8s 入口提供升级引导，不承诺节点管理模块内的 K8s 部署脚本。
- 校准节点与安装规则：补充默认云区域禁止编辑、新建云区域自动初始化服务与变量模板、节点活跃判定、批量控制器安装入口限制与 Windows 手动安装口径。
- 校准配置与安装包规则：补充同一节点同一采集器仅保留一个生效主配置、子配置排序规则，以及安装包按文件名识别版本且重复版本不可重复上传。
- 补充 Sidecar 手动安装脚本、Windows 安装配置与令牌下载能力描述。
