# CMDB 产品PRD

版本：v1.2

日期：2026-04-24

## 1. 背景与目标

CMDB 是资产与配置数据的统一管理中心，围绕“模型驱动的资产管理”与“自动发现采集”两条主线，提供资产建模、实例维护、关系管理、检索与变更追踪能力。

目标：
- 统一资产模型定义与关系约束，降低资产管理成本。
- 支撑资产实例全生命周期管理与可追溯变更记录。
- 通过自动发现提升数据采集效率与准确性。

## 2. 目标用户与使用场景

- 资产管理员：负责模型配置、字段分组、资产结构维护。
- 运维工程师：维护资产实例、关联关系、搜索与排障。
- 平台管理员：管理自动发现任务、采集插件与 SOID 库。

## 3. 范围定义

CMDB 提供以下能力范围：
- 模型管理：模型分类、模型、属性、字段分组、模型关系、联合唯一规则、自动关联规则、模型配置导入导出。
- 资产管理：资产实例增删改查、批量导入、批量导出、资产关联、详情查看。
- 资产详情：基础信息、关联关系、变更记录、配置文件版本。
- 资产视图：按模型/分类汇总与快速跳转。
- 资产检索：全文检索、按模型统计与分页查看。
- 自动发现：采集对象树、采集任务管理、执行状态汇总、采集插件说明、SOID 库。
- 变更记录：变更日志与时间筛选。
- 数据订阅：围绕模型与实例配置订阅规则、接收对象与通知渠道，并按组织范围控制可见与可管理边界。
- 个人配置：展示字段与用户保存的筛选条件等个性化配置。
- 公共选项库：为枚举类属性提供可复用的选项集合，并支持查看引用关系。

## 5. 功能需求

### 5.1 模型管理

- 模型分类管理：新增、修改、删除、列表展示。
- 模型管理：新增、修改、删除、复制、列表与详情。
- 属性管理：新增与配置模型属性，内置默认属性 `inst_name` 与 `organization`。
- 枚举属性：支持单选与多选模式，并可复用公共选项库。
- 枚举规则：枚举字段创建后，选项来源规则与单选/多选模式保持稳定，不支持直接切换。
- 字段分组：可创建/编辑/删除字段分组，支持分组排序与折叠状态。
- 关系管理：定义模型间关联关系，支持关联类型与映射约束。
- 联合唯一规则：支持为模型配置单字段或多字段组合唯一规则。
- 自动关联规则：支持为模型关系配置自动关联规则，并按规则自动维护实例关系。
- 模型配置导入导出：支持模型配置导入与导出，用于模板复用与跨环境迁移。

### 5.2 资产管理

- 资产实例：
  - 列表查询支持筛选、分页、排序。
  - 实例新增、修改、删除（含批量操作）。
  - 支持导入模板下载、Excel 导入与按字段/关联导出。
- 资产关联：
  - 创建/取消资产间关联关系。
  - 支持按关联类型查看与管理。
- 资产详情：
  - 查看基础属性、关联关系与变更记录。
  - 支持轻量拓扑视图展示，并支持按层展开与节点扩展。

### 5.3 资产详情与配置文件版本

- 配置文件列表：支持按实例查看已采集的配置文件清单。
- 版本历史：支持按文件查看版本列表、采集状态与生成时间。
- 内容查看：支持按编码查看配置文件内容。
- 版本对比：支持选择两个版本进行差异对比。

### 5.4 资产视图

- 按模型分类展示资产概览卡片。
- 支持关键字搜索并快速跳转到对应资产列表页。

### 5.5 资产检索

- 全文检索：输入关键字检索资产实例。
- 搜索结果：按模型分组统计与分页展示。
- 支持大小写敏感配置与本地搜索历史保存。

### 5.6 自动发现与采集任务

- 采集对象树：按类型展示可采集对象与标签。
- 采集任务管理：新增、编辑、删除、列表与详情。
- 采集任务配置：支持周期执行、接入点、凭据与超时设置。
- 采集任务执行：支持手动触发任务执行，并查看任务执行状态汇总。
- 采集任务对象选择：支持节点、模型实例和云区域等接入对象选择。
- 采集结果摘要：支持新增/更新/删除/关联的分类数据摘要。
- 采集文档：按采集对象查看对应的插件说明文档。
- 采集任务映射：支持在实例侧查看相关采集任务名称与跳转入口。

### 5.7 SOID 特征库

- 维护设备 `sysObjectID` 映射关系。
- 支持新增、编辑、删除、列表展示。

### 5.8 变更记录

- 变更日志列表与详情查看。
- 支持按时间范围筛选与按类型展示。

### 5.9 数据订阅

- 支持在资产场景下按模型配置订阅规则。
- 订阅规则支持启用、停用、编辑、删除与分页查询。
- 支持两种筛选方式：按条件筛选目标实例，或直接选择目标实例。
- 支持三类触发条件：属性变更、关联关系变更、到期提醒。
- 关联关系变更支持在单条规则中配置多个关联模型，并分别维护监听字段。
- 支持配置接收对象与通知渠道。
- 订阅规则支持在当前组织及其下级组织范围内查看，仅所属组织可执行管理操作。

### 5.10 个人配置

- 用户级别的展示字段配置。
- 用户保存的筛选条件与其他个性化配置。
- 支持按模型保存和复用常用筛选条件。

### 5.11 公共选项库

- 支持公共选项库的新增、编辑、删除与列表查询。
- 支持枚举属性引用公共选项库，统一选项定义与复用。
- 支持查看公共选项库的引用模型与属性。
- 公共选项库被引用时不允许删除。
- 公共选项库选项变更后，已引用该选项库的枚举属性保持同步。

### 5.12 权限规则

- 模型、实例、采集任务按组织与对象权限控制可见范围和可操作范围。
- 订阅规则按当前组织范围控制可见范围，按所属组织控制管理范围。
- 默认组织下模型默认可查看，操作权限按组织权限配置生效。

## 6. 关键业务流程（摘要）

- 模型新增与配置：创建分类 → 新增模型 → 配置属性与字段分组 → 设置关系。
- 资产新增与维护：选择模型 → 新增实例 → 编辑/导入 → 维护关联关系。
- 配置文件追踪：选择实例 → 查看配置文件列表 → 打开版本内容或对比差异。
- 资产检索：输入关键字 → 查看模型统计 → 按模型查看详情。
- 自动发现：选择采集对象 → 配置任务 → 执行采集 → 查看采集摘要。
- 数据订阅：选择模型或实例范围 → 配置触发条件与通知对象 → 启用规则。

## 7. 数据结构与字段字典

### 7.1 模型与实例默认属性

- `inst_name`：实例名（唯一、必填、可编辑）。
- `organization`：所属组织（必填、可编辑）。

### 7.2 属性类型

- 字符串：`str`
- 整数：`int`
- 枚举：`enum`
- 时间：`time`
- 用户：`user`
- 组织：`organization`
- 布尔：`bool`
- 密码：`pwd`
- 标签：`tag`
- 表格：`table`

枚举模式：
- 单选：`single`
- 多选：`multiple`

### 7.3 属性约束

- `is_only`：唯一
- `is_required`：必填
- `editable`：可编辑
- `enum_select_mode`：枚举模式（单选/多选）
- `enum_rule_type`：枚举规则类型（内置/公共选项库）
- `public_library_id`：公共选项库引用ID
- `unique_rules`：联合唯一规则

### 7.4 关联类型

- `belong`（属于）
- `group`（组成）
- `run`（运行于）
- `install_on`（安装于）
- `contains`（包含）
- `connect`（关联）

### 7.5 关联映射约束

- `1:1`：一对一
- `1:n`：一对多
- `n:1`：多对一
- `n:n`：多对多

### 7.6 自动发现采集类型

采集插件类型：
- `vm`、`snmp`、`k8s`、`cloud`、`protocol`、`host`、`db`、`middleware`、`ip`、`other`

采集驱动类型：
- `protocol`（协议采集）
- `job`（脚本采集）

采集对象树（示例范围，实际以配置为准）：
- 容器（K8S、Docker）
- VMware（vCenter）
- 网络设备（SNMP）
- 数据库（MySQL、PostgreSQL、Redis）
- 云平台（阿里云、腾讯云）
- 主机（主机、物理服务器）
- 中间件（Nginx、Zookeeper、Kafka、Consul、Etcd、RabbitMQ、Tomcat）

### 7.7 采集任务核心字段（摘要）

- `name`：任务名称
- `task_type`：任务类型
- `driver_type`：驱动类型
- `model_id`：模型ID
- `is_interval`、`scan_cycle`：周期配置
- `access_point`：接入点
- `credential`：凭据（加密存储）
- `timeout`：单对象超时
- `exec_status`、`exec_time`：执行状态与时间
- `input_method`：录入方式
- `data_cleanup_strategy`：数据清理策略
- `collect_digest`：采集摘要统计
- `format_data`：采集摘要数据（新增/更新/删除/关联）

### 7.8 订阅规则核心字段（摘要）

- `name`：规则名称
- `organization`：所属组织
- `model_id`：目标模型
- `filter_type`：筛选方式（条件 / 指定实例）
- `instance_filter`：实例筛选条件
- `trigger_types`：触发类型列表
- `trigger_config`：触发条件配置
- `trigger_config.relation_change.related_models`：关联变化监听模型集合
- `recipients`：接收对象
- `channel_ids`：通知渠道
- `is_enabled`：启用状态

### 7.9 SOID 映射字段（摘要）

- `model`：设备型号
- `oid`：设备 OID（唯一）
- `brand`：品牌
- `device_type`：设备类型
- `built_in`：是否内置

### 7.10 配置文件版本核心字段（摘要）

- `instance_id`：关联实例ID
- `collect_task_id`：来源采集任务ID
- `file_path`：配置文件路径
- `version`：版本号
- `status`：采集结果状态
- `created_at`：版本生成时间

## 8. 需求优先级（MoSCoW）

- Must：模型管理、资产实例管理、资产关联、全文检索、采集任务管理、变更记录。
- Should：资产视图、批量导入导出、采集文档、SOID 库、数据订阅、联合唯一规则、自动关联规则。
- Could：个人配置、字段分组排序优化、模型复制增强。
- Won’t：跨模块联动通知、可配置审批流。

## 9. 验收标准（按场景）

- 新增模型场景：
  - 可创建分类与模型，模型可出现在列表并可进入详情配置。
- 配置模型属性场景：
  - 新增/编辑属性后，实例新增表单与列表字段同步生效。
  - 枚举属性支持单选或多选，并可引用公共选项库。
  - 支持配置联合唯一规则与自动关联规则。
- 采集任务场景：
  - 可创建采集任务并配置接入点/凭据/周期/超时，任务可正常保存并出现在列表。
  - 可在实例侧查看相关采集任务名称。
- 配置文件版本场景：
  - 可按实例查看配置文件列表与版本历史。
  - 可查看指定版本内容，并支持两个版本的差异对比。
- 资产搜索场景：
  - 输入关键字可返回模型统计与分页列表，支持大小写敏感切换。
- 资产关联场景：
  - 可选择关联类型并创建关联关系，关联关系可在列表与拓扑中查看。
- 拓扑查看场景：
  - 资产详情页可查看当前实例关联范围内的拓扑结构，并支持按层展开与节点扩展。
- 变更记录场景：
  - 变更日志支持查看详情与时间筛选。
- 批量导入/导出场景：
  - 可按模型导入实例数据，导出支持选字段与关联信息。
  - 支持模型配置导入与导出。
- 数据订阅场景：
  - 可创建订阅规则并配置实例范围、触发条件、接收对象与通知渠道。
  - 规则支持启用、停用、编辑与删除。
  - 关联变化触发支持配置多个关联模型。
- 公共选项库场景：
  - 可维护公共选项库并被枚举属性复用，选项变更后可同步生效。
  - 可查看公共选项库的引用关系，已被引用的选项库不可删除。
