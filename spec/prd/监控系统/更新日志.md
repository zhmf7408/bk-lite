# 监控系统 PRD 更新日志

## 2026-04-06（v1.5）

- 文档版本升级为 `v1.5`（日期 `2026-04-06`）。
- 校准 Integration 范围：补充正式子导航“对象”，纳入监控对象类型与监控对象治理口径。
- 校准接入口径：补充 API 模板接入引导页，以及自建模板创建、编辑、删除能力。
- 校准验收口径：Integration 验收补充 object 子页路由、对象排序与可见性控制。

## 2026-03-30（v1.4）

- 文档版本升级为 `v1.4`（日期 `2026-03-30`）。
- 校准 View 口径：明确蜂巢视图仅对 `Pod`、`Node` 对象开放，其余对象使用列表视图。
- 校准 Search 口径：明确单位展示为后端自动换算结果。
- 校准 Integration 口径：收敛为当前可用配置入口，非 `K8s` 对象使用自动配置页，`K8s` 对象使用专用三步向导；移除导入/导出与手动接入的现状表述。

## 2026-03-23（v1.3）

- 文档版本升级为 `v1.3`（日期 `2026-03-23`）。
- 校准 Event 规则：明确 `info` 事件用于恢复判断，不参与告警升级。
- 校准通知口径：明确仅非 `info` 事件进入通知发送链路。

## 2026-03-18（v1.2）

- 文档版本升级为 `v1.2`（日期 `2026-03-18`）。
- 校准 View 口径：补充列表 / 蜂巢双视图、实例查看弹层（监控视图 / 相关告警）与实例详情指标页的入口边界。
- 校准 Search 口径：补充多查询组、维度条件 AND 组合、趋势图与维度表展示、命名查询保存与加载。
- 校准 Event 规则：区分策略启用告警类型（`threshold` / `no_data`）与告警实体类型（`alert` / `no_data`），补充 `info` 事件与停用策略自动关闭活跃告警规则。
- 校准 Integration 与通知口径：补充自动 / 手动接入、模板配置状态、系统管理通知渠道来源与 NATS 渠道规则。
- 删除 PRD 正文中的待确认项，收敛为当前稳定状态描述。

## 2026-02-27（v1.1）

- 文档版本升级为 `v1.1`（日期 `2026-02-27`）。
- 告警名称模板变量完成调整：
	- 可选变量由包含`${instance_name}`、`${instance_id}`的旧集合，调整为以`${resource_name}`、`${dimension_value}`为主的新集合。
	- `${resource_name}`用于承载原“实例名+维度”中的实例名部分，统一为仅实例名称。
	- `${dimension_value}`用于承载维度信息，统一格式为`维度名称:维度值`并使用英文逗号分隔。
- PRD正文同步收敛为“最终状态描述”，不再记录“隐藏/新增”等过程语句。
