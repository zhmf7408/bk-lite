# 产品介绍

## 产品简介
系统管理（System Management）是 BK-Lite 平台的统一底座与治理中枢。它致力于为企业提供全局范围内的身份认证、组织架构、权限管控、通知渠道与安全审计能力。通过灵活的菜单编排与细粒度的数据权限控制，系统管理模块帮助企业构建起既高度协作又安全隔离的多应用治理体系。

## 核心优势
*   **统一的身份与组织治理**：集中管理全公司的组织架构谱系与员工账号，支持与企业级认证源同步，告别各系统分散建设账号的混乱局面。
*   **极高扩展性的权限引擎**：基于“组织-用户-角色”的多维权限与自定义菜单组合，支持应用间的数据权限精细化控制，有效应对复杂企业架构下的按需授权。
*   **企业级安全防线**：内置全方位的密码策略、登录控制设置，并将核心敏感凭据（如加密通道 API 密钥等）严密接管，构筑可靠的防御阵地。
*   **全链路操作审计**：提供从用户登录、关键操作到系统异常的详尽日志跟踪与导出功能，让每一次风险操作都有迹可循。

## 应用场景
*   **统一办公入口搭建**：当企业需要将多个分散的内部系统合并到一个平台上时，系统管理可作为统一管理底座，一站式分发不同部门员工的菜单可见性与菜单功能结构。
*   **高合规审计与日志追溯**：在金融、政务等对合规要求极高的行业中，审计人员可以直接在后台实时查询或导出详细的操作轨迹与错误日志，用于安全检查或责任定界。
*   **全平台消息通道建设**：IT 管控部门可统一将企业微信、钉钉等消息平台信息绑定在此系统下，为上层的各种系统（如监控、OpsPilot等）提供统一的对外通知发送渠道。
