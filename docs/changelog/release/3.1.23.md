# 3.1.23 版本日志
版本发布时间：2026年4月3日
了解更多产品能力，欢迎查阅[官方文档](https://www.bklite.ai)或直接体验[Demo环境](https://bklite.canway.net)

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 监控中心 | 开放监控对象管理功能 |
| 监控中心 | 支持通过 API（PULL / PUSH 模式）自定义上报监控数据 |
| CMDB | 模型配置支持导入与导出 |
| OpsPilot | 新增工具：Agent Browser |

### 功能优化

| 模块 | 功能优化 |
| --- | --- |
| 节点管理 | 节点默认开启多类日志采集探针组件 |
| 日志中心 | 优化日志搜索体验：支持根据搜索结果自动过滤并展示相关字段 |
| CMDB | 配置采集任务支持自动清理下架资产 |
| CMDB | 模型支持多字段联合唯一性校验 |
| OpsPilot | 工作台支持在线测试 |
