# V3.1.10 版本日志
版本发布时间：2025年12月12日
了解更多产品能力，欢迎查阅[官方文档](https://www.bklite.ai)或直接体验[Demo环境](https://bklite.canway.net)

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 节点管理 | 新增：支持 Windows 节点手动安装控制器 |
| CMDB | 支持通过 SSH 自动采集物理服务器硬件与资源指标（CPU / 内存 / 磁盘等） |
| OpsPilot | 机器人新增图片能力支持 |
