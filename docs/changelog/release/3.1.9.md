# 3.1.9 版本日志
版本发布时间：2025年12月5日
了解更多产品能力，欢迎查阅[官方文档](https://www.bklite.ai)或直接体验[Demo环境](https://bklite.canway.net)

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| CMDB | 配置采集现已支持查看 全部分类 |
| CMDB | 插件卡片新增 任务概览 展示 |
| CMDB | 采集任务详情新增 概览统计 |
| 监控系统 | 新增 K8s 监控接入 |
| OpsPilot | Chatflow 应用新增三种触发方式：移动端、嵌入对话、Web |
| OpsPilot | 移动端能力上线，可从移动端入口发起智能对话并查看历史记录 |
| OpsPilot | Web 端接入，支持应用查询与对话交互 |
