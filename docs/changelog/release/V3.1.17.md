# V3.1.17 版本日志

了解更多产品能力，欢迎查阅[官方文档](https://www.bklite.ai)或直接体验[Demo环境](https://bklite.canway.net)

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| CMDB | 模型属性类型新增约束条件支持： |
| CMDB | string 支持正则校验 |
| CMDB | number 支持最大 / 最小值范围约束 |
| CMDB | time 支持日期与时间格式 |
| CMDB | 模型属性支持添加用户提示信息 |
| 监控系统 | 新增无数据告警基准机制，支持维度级别精准识别无数据事件 |
| OpsPilot | 日志支持查看错误原因并定位错误节点 |
