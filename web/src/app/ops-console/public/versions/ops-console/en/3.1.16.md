# 3.1.16 Release Notes
Release Date: January 30, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Node Management | The editor window now supports full-screen mode. |
| CMDB | Added saved search conditions. |
| Alert Center | Introduced intelligent noise reduction to automatically identify and merge Event alerts across multiple dimensions. |
| OpsPilot | Knowledge bases now support image uploads. |
| OpsPilot | Supports viewing document processing failure reasons. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Fixed known issues where nodes reverted automatically after modification. |
| CMDB | Merged network device and topology collection tasks, allowing users to choose whether to collect relationship data. |
| Monitoring | Metric views now automatically adapt the best unit for the 95th percentile. |
| Monitoring | Optimized the monitoring policy configuration workflow. |
| Monitoring | Fixed issues with metric management retrieval. |
