# 3.1.11 Release Notes
Release Date: December 19, 2025
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Node Management | Added a controller version management mechanism with version display in the node list. |
| Node Management | Optimized the node details page, supporting sub-configuration search and deletion. |
| CMDB | Added multi-condition filtering to the asset page. |
| CMDB | Added built-in data sources for operations analytics, supporting alert trends, log hit counts, and single metric queries. |
| Monitoring | Added Tencent Cloud resource monitoring. |
| OpsPilot | Added an intent recognition node to WorkFlow. |

### Improvements

| Module | Improvements |
| --- | --- |
| CMDB | Enhanced VCenter collection capabilities along with multiple related optimizations. |
