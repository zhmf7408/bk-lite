# 3.1.17 Release Notes
Release Date: February 6, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Added support for constraints on model attribute types: |
| CMDB | string supports regex validation |
| CMDB | number supports minimum and maximum range constraints |
| CMDB | time supports date and time formats |
| CMDB | Model attributes now support user guidance text. |
| Monitoring | Added a no-data alert baseline mechanism for precise no-data detection at the dimension level. |
| OpsPilot | Logs now support viewing error causes and locating failed nodes. |

### Improvements

| Module | Improvements |
| --- | --- |
| Monitoring | Optimized the K8s metric system and improved alert list display of alert types and related capabilities. |
| Monitoring | Fixed issues with Kafka monitoring SASL parameter construction and MySQL monitoring credentials containing special characters. |
| OpsPilot | The browser_use tool now automatically captures key screenshots during browser operations, making agent actions visible. |
| OpsPilot | Configuration documentation for the PostgresDB, K8s, and Web application inspection assistants has been updated on the official website. |
