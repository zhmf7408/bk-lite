# 3.1.8 Release Notes
Release Date: November 28, 2025
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Node Management | On the cloud region home page, users can deploy cloud regions in container mode and view Nats Executor and Stargazer status cards. |
| CMDB | Added raw data viewing in configuration collection task details to help diagnose credential and network-related collection failures, excluding K8s, cloud platforms, and VC. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Removed the separate configuration page from Node Management > Cloud Region > Nodes and merged it into the node details drawer. |
| CMDB | Configuration collection task details now support viewing exception logs for individual records written to CMDB. |
