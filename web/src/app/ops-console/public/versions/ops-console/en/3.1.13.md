# 3.1.13 Release Notes
Release Date: January 9, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Node Management | Supports remote installation of Linux nodes using key-based credentials. |
| CMDB | Added field grouping capability to model management. |
| CMDB | Added Windows Server support to configuration collection. |
| Monitoring | Added automatic metric unit conversion. |
| OpsPilot | Added the BrowserUse tool, enabling agents to operate browsers for automated inspection tasks. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Optimized error message display for managed components. |
| Monitoring | Fixed monitoring issues caused by passwords containing special characters. |
| OpsPilot | Enhanced knowledge base recognition of images embedded in documents. |
