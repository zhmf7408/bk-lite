# 3.1.23 Release Notes
Release Date: April 3, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Monitoring | Opened monitoring object management capabilities. |
| Monitoring | Supports custom monitoring data reporting through APIs in both PULL and PUSH modes. |
| CMDB | Model configurations now support import and export. |
| OpsPilot | Added the Agent Browser tool. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Multiple log collection probe components are now enabled by default on nodes. |
| Log Center | Optimized log search by automatically filtering and displaying relevant fields from search results. |
| CMDB | Configuration collection tasks now support automatic cleanup of decommissioned assets. |
| CMDB | Models now support combined uniqueness validation across multiple fields. |
| OpsPilot | The workbench now supports online testing. |
