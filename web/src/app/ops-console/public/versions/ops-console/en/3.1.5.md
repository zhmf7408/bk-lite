# 3.1.5 Release Notes
Release Date: October 31, 2025
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| OpsPilot | Added new Chatflow trigger types, including WeCom, DingTalk, and WeChat Official Accounts, expanding automation scenarios. |
| System Management | Added a super administrator capability for centralized permission and configuration management. |
| System Management | Added virtual teams for flexible team organization and collaboration. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Optimized probe management by separating collectors and controllers for a clearer structure and easier maintenance. |
| Node Management | Optimized cloud region interactions and UI for node and configuration management pages. |
