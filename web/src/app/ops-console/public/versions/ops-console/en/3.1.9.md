# 3.1.9 Release Notes
Release Date: December 5, 2025
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Configuration collection now supports viewing all categories. |
| CMDB | Added a task overview display to plugin cards. |
| CMDB | Added overview statistics to collection task details. |
| Monitoring | Added K8s monitoring onboarding. |
| OpsPilot | Chatflow applications now support three trigger methods: mobile, embedded chat, and web. |
| OpsPilot | Mobile support is now available, allowing users to start intelligent conversations and view history from the mobile entry. |
| OpsPilot | Added web access, supporting application lookup and conversational interaction. |

### Improvements

| Module | Improvements |
| --- | --- |
| CMDB | Collection tasks now capture and display sync data exception logs. |
| Log Center | Optimized Syslog log ingestion. |
