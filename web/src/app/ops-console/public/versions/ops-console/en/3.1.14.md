# 3.1.14 Release Notes
Release Date: January 16, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Added model duplication capability. |
| CMDB | Full-text search now supports case-insensitive fuzzy matching and fields that map display names to stored values, including enumerations, users, and organizations. |
| CMDB | Instance topology now supports continuous expansion to the next level down. |
| OpsPilot | Added built-in PostgresDB and K8s inspection assistants for ReAct scenario validation. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Optimized the cloud region deployment workflow to improve node deployment efficiency. |
| Node Management | Fixed issues with incorrect node type identification. |
| Monitoring | Comprehensively optimized the monitoring metric system to improve metric value and accuracy. |
| Monitoring | Fixed VMware monitoring collection issues. |
| OpsPilot | Scheduled task execution times now prioritize the user-configured time zone. |
| OpsPilot | Optimized K8S tool configuration reading by passing parameters directly instead of reading from a path. |
