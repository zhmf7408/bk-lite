# 3.1.10 Release Notes
Release Date: December 12, 2025
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Node Management | Added support for manual installation of Windows node controllers. |
| CMDB | Supports automatic collection of physical server hardware and resource metrics via SSH, including CPU, memory, and disk. |
| OpsPilot | Robots now support image capabilities. |

### Improvements

| Module | Improvements |
| --- | --- |
| Platform | Optimized bulk import and export workflows. |
| Monitoring | Comprehensively tuned onboarding capabilities for various monitoring objects. |
| Log Center | Optimized Docker log ingestion. |
