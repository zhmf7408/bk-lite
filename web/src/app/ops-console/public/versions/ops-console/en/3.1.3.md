# 3.1.3 Release Notes
Release Date: September 26, 2025
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| System Management | Added Tencent BlueKing as an identity source, supporting SSO with the BlueKing platform and user synchronization. |
| OpsPilot | Chatflow applications now support testing, allowing immediate feedback on results. |
| OpsPilot | Added history viewing for Chatflow applications. |
| Log Center | Added a time distribution chart to the log alert event list. |
| Operations Analytics | Supports 2.5D topology visualization for full application architectures. |
