# 3.1.7 Release Notes
Release Date: November 21, 2025
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| OpsPilot | Added a Web chat dialog for Chatflow, supporting AG-UI protocol integration with other systems and multi-session management. |
| System Management | Added error log management for faster filtering and troubleshooting. |
| System Management | Added password settings with support for strong password policies to improve account security. |
| Log Center | Added SNMP Trap log ingestion for real-time alerting and analysis. |
| Monitoring | Supports bulk importing monitoring collection configurations from Excel. |
