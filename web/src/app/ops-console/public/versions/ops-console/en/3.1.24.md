# 3.1.24 Release Notes
Release Date: April 10, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| System Management | Supports inheriting parent organization roles to child organizations. |
| CMDB | Added automatic asset association, allowing asset relationships to be created automatically based on rules. |
| CMDB | Data subscriptions now support trigger conditions based on model relationships. |
| Operations Analytics | Supports importing and exporting dashboards, topology or architecture views, data sources, and namespaces. |
| Operations Analytics | Added a table-type component. |

### Improvements

| Module | Improvements |
| --- | --- |
| Monitoring | Fixed an issue where custom monitoring reports could not automatically create assets. |
