# 3.1.15 Release Notes
Release Date: January 23, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Added Postgres and Docker asset collection capabilities. |
| CMDB | Supports creating models directly under model groups, improving the interaction experience. |

### Improvements

| Module | Improvements |
| --- | --- |
| CMDB | Fixed version compatibility issues with agent-side dependencies for configuration collection. |
| Monitoring | Enhanced the monitoring policy page with metric previews, referenced variables, and other usability improvements. |
| Monitoring | Optimized the display of key metrics in the monitoring view list. |
| Log Center | Updated the Vector collection component to support all Linux distributions. |
| OpsPilot | Added built-in PostgresDB, K8s, and Web application inspection assistants that can be used after configuration. |
| OpsPilot | Optimized the knowledge base training workflow and interaction feedback, and improved OCR model permission guidance for a better overall experience. |
| OpsPilot | Fixed WeCom notification issues and node execution issues in the workbench. |
