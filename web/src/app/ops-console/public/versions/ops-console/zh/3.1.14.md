# 3.1.14 版本日志
版本发布时间：2026年1月16日
了解更多产品能力，欢迎查阅[官方文档](https://www.bklite.ai)或直接体验[Demo环境](https://bklite.canway.net)

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| CMDB | 新增 模型复制功能 |
| CMDB | 全文检索支持 大小写模糊匹配，支持枚举、用户、组织等显示名与存储数据映射的字段 |
| CMDB | 实例拓扑支持 持续向下一级展开 |
| OpsPilot | 新增 PostgresDB 巡检助手、K8s巡检助手（ReAct 场景验证） |

### 功能优化

| 模块 | 功能优化 |
| --- | --- |
| 节点管理 | 优化 云区域部署操作路径，提升节点部署效率 |
| 节点管理 | 修复 节点类型识别错误 等问题 |
| 监控系统 | 监控指标体系全面优化调整，提升指标价值度和准确性 |
| 监控系统 | 修复 VMware 监控采集异常 等问题 |
| OpsPilot | 定时任务执行时间调整为 优先读取用户设置的时区 |
| OpsPilot | 优化 K8S 工具配置读取方式，由 path 读取改为参数直接传入 |
