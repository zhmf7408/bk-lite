# 3.1.11 版本日志
版本发布时间：2025年12月19日
了解更多产品能力，欢迎查阅[官方文档](https://www.bklite.ai)或直接体验[Demo环境](https://bklite.canway.net)

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 节点管理 | 新增 控制器版本号管理机制，并支持在节点列表中展示 |
| 节点管理 | 节点详情页体验优化，支持 子配置搜索与删除 等操作 |
| CMDB | 资产页面新增 多条件筛选能力 |
| CMDB | 运营分析内置数据源，支持 告警趋势、日志命中数、单监控指标查询 |
| 监控系统 | 新增 腾讯云资源监控能力 |
| OpsPilot | WorkFlow新增 意图识别节点 |
